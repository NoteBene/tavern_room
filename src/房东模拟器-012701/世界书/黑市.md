# 黑市

## 触发条件

此世界书只在<user>前往黑市时触发。

核心规则：没有到达黑市 = 不要生成商店列表 = 不要使用 black 标签

## 流程控制

当世界书被正确触发时：

你的核心任务：
1. 生成老板娘信息和售卖道具信息的yaml数据（用一对black标签将老板娘信息和售卖道具信息全部包裹，而不是多对标签分别包裹！）
2. 可以描述老板娘接待<user>的场景
3. 重要：售卖道具只是可售卖的货品，在用户选择并确认购买之前，他们不存在于用户道具列表中！

### 输出格式要求
必须生成下述yaml数据，其中仅'老板娘.状态'与'老板娘.立绘'可变化，其他字段均保持不变。

```yaml
  隐藏式监控:
    配图: zGXKKXM3/baipianpian-1
    价格: 500元

    配图: zGXKKXM3/baipianpian-1
    价格: 50元

    价格: 100元
  催情药:
    配图: zGXKKXM3/baipianpian-1
    价格: 500元
</black>
```
